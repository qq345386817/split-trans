PhraseCut कैसे इस्तेमाल करें

यह app sentence-based language learning के लिए बनाया गया है। मुख्य workflow है: एक वाक्य analyze करें, useful items save करें, बाद में review करें और study materials export करें।

1. एक वाक्य से सीखें

  1. Learn tab खोलें।
  2. जिस वाक्य को समझना है उसे paste या type करें।
  3. जरूरत हो तो source और target language चुनें।
  4. Analysis चलाकर full translation और word/phrase meaning देखें।

Learn screen एक समय में एक वाक्य पर focus करता है: पहले full meaning समझें, फिर important words/phrases देखें, और आखिर में review लायक content save करें।

Learn screen किसलिए है?

PhraseCut isolated word lists के बजाय real sentences से सीखने में मदद करता है। App पहले sentence meaning दिखाता है, फिर useful parts जिन्हें आप later review के लिए रख सकते हैं।

2. उपयोगी words, phrases या sentences save करें

जब कुछ रखने लायक मिले, उसे study library में save करें। App item के साथ उसका source sentence और context meaning भी रख सकता है।

3. Saved items review करें

Review tab खोलकर saved items फिर देखें। इससे useful vocabulary भूलने से पहले वापस active होती है।

4. Study library manage करें

Library tab में real reading और study sessions से collected words, phrases और sentences browse करें।

5. Study materials export करें

CSV या Anki-compatible files में library भेजने के लिए export options इस्तेमाल करें।
