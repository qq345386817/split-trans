如何使用 Sentwise

這是一款面向語言學習者的句子優先 App。你可以先分析一句話，再保存值得學習的內容，之後複習並匯出。

1. 從一句話開始學習

  1. 打開 Learn 頁面。
  2. 貼上或輸入你想看懂的句子。
  3. 如有需要，選擇來源語言和目標語言。
  4. 開始分析，查看整句翻譯以及詞語或短語含義。

Learn 頁的核心順序是：先理解整句意思，再看值得學習的詞語或短語，最後把真正有價值的內容保存下來複習。

Learn 頁面是做什麼的？

Sentwise 不是要你機械背單詞，而是希望你透過真實句子理解表達，再從句子裡提煉出值得反覆學習的內容。

2. 保存有價值的單詞、短語或句子

當你遇到值得記住的內容時，可以保存到學習庫。除了內容本身，App 還會保留它所在的句子和語境意思。

3. 繼續複習

打開 Review 頁面，回看你之前保存的內容，在忘記前重新激活這些表達。

4. 管理學習庫

在 Library 頁面中，你可以統一查看自己累積下來的單詞、短語和句子。

5. 匯出學習資料

你可以把學習庫匯出為 CSV 或 Anki 相容檔案，方便後續整理，或匯入到自己常用的學習工具。
