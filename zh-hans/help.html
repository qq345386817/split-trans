使用 PhraseCut

PhraseCut 让学习流程保持聚焦：理解一个真实句子，保存有用片段，在遗忘前复习。

1. Analyze one sentence

Open Learn, paste or type a sentence, choose the language pair, then run the analysis. Start with the full meaning before studying individual words.

2. Save useful items

Save a full sentence, word, or phrase when it is worth revisiting. PhraseCut keeps the original sentence and translation with the saved item.

3. Review before you forget

Open Review to revisit due items. The review flow is intentionally lightweight so it fits into short reading sessions.

4. Export your library

Use Library export when you want CSV or Anki-compatible files for another study workflow.
