如何使用 Sentwise

这是一款面向语言学习者的句子优先 App。你可以先分析一句话，再保存值得学习的内容，之后复习并导出。

1. 从一句话开始学习

  1. 打开 Learn 页面。
  2. 粘贴或输入你想看懂的句子。
  3. 如有需要，选择源语言和目标语言。
  4. 开始分析，查看整句翻译以及词语或短语含义。

Learn 页的核心顺序是：先理解整句意思，再看值得学习的词语或短语，最后把真正有价值的内容保存下来复习。

Learn 页面是做什么的？

Sentwise 不是让你机械背单词，而是希望你通过真实句子理解表达，再从句子里提炼出值得反复学习的内容。

2. 保存有价值的单词、短语或句子

当你遇到值得记住的内容时，可以保存到学习库。除了内容本身，App 还会保留它所在的句子和语境意思。

3. 继续复习

打开 Review 页面，回看你之前保存的内容，在忘记前重新激活这些表达。

4. 管理学习库

在 Library 页面中，你可以统一查看自己积累下来的单词、短语和句子。

5. 导出学习资料

你可以把学习库导出为 CSV 或 Anki 兼容文件，方便后续整理，或导入到自己常用的学习工具。
